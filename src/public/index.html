Documentação de rotas

    GET

  • /

    Index com descrição das rotas

  • /api

    Retorna Nome e versão da API

  • /projects

    Retorna projetos cadastrados

  • /projects/titulo do projeto

    Retorna projeto cadastrado passando título pela url

  • /tasks

    Retorna Tarefas cadastradas


    PUT

  • /projects/id

    Atualiza projeto passando id pela url e o resto das informações pelo body

  • /tasks/id

    Atualiza tarefa passando id da tarefa pela url


    POST

  • /users

    Cadastra Usuario e envia notificação por email

  • /users/authenticate

    Consulta usuario no banco e retorna um token caso usuario exista

  • /users/refresh-token

    Verifica se tem algum token válido no body ou header ou na url caso tenha renova o token por mais um dia sem que o usuário precise logar novamente

  • /projects

    Cadastra projeto

  • /tasks

    Cadastra tarefa passando ou não o id do projeto


    DELETE

  • /projects/id

    Exclui a tarefa passando id da mesma na url

  • /tasks/id

    Exclui a tarefa passando id da mesma na url
